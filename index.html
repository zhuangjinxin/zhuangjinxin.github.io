面试汇总:《Redis》

发表于 2018-03-21 | 分类于 面试

为什么Redis性能高？

  • 纯内存存储
  • I/O多路复用
  • 单线程架构

Redis常用数据类型

  • 字符串 string
  • 哈希 Hash
  • 列表 list
  • 集合 set
  • 有序集合 sorted set(zset)

常用命令

  • string: set get mset mget del incr decr append strlen setrange getrange
  • hash: hset hget hmset hmget hdel hgetall hexists hkeys hvalues hsetnx hincrby hstrlen
  • list: lpush rpush lpop rpop blpop brpop lrange lindex llen lset
  • set: sadd srem scard smermbers sismember srandmember spop|sinter sunion sdiff
  • zset: zadd zrem zcard zincrby zrank zrevrank zrange zrangebyscore zinterstore zunionstore
  • 发布订阅: publish subscribe unsubscribe psubscribe pubsub
  • 键管理: type object exists dbsize rename renamenx randomkey
  • 过期时间: expire expireat pexpire pexpireat ttl pttl persist
  • 迁移: move dump+restore migrate
  • 遍历键: keys scan
  • 数据库: select flushdb flushall

面试汇总:《Java》

发表于 2018-03-21 | 分类于 面试

集合

Java中常用的数据结构

  • List: ArrayList、LinkedList
  • Set: HashSet
  • Map: HashMap、ConcurrentHashMap

ArrayList、LinkedList

  • ArrayList：基于动态数组实现 默认大小10 扩容1.5倍+1 线程不安全 CopyOnWriteArrayList或Vector
  • LinkedList：基于双向链表实现
  • 区别：
    1. 随机访问元素时，ArrayList优于LinkedList；(因为LinkedList要移动指针)
    2. 新增或删除元素时，LinkedList优于ArrayList；（因为ArrayList要移动数据）
  • 注意：Arraylist不能用正向for循环删除元素
