数据库原理

三大范式

第一范式： 表中的每一列都是不可拆分的最小单元
第二范式： 表中的每一列必须依赖于主键，不能与主键没有关系
第三范式： 每个非主属性都必须与主键直接相关，而不间接相关

五大约束

PRIMARY KEY： 主键约束
UNIQUE： 唯一性约束
DEFAULT： 默认值
NOT NULL： 非空约束
FOREIGN KEY： 外键约束

面试汇总:《Spring》

发表于 2018-03-21 | 更新于 2018-03-27 | 分类于 面试

Spring

Spring是一套轻量级的、企业级的Java开发框架。

AOP

概念：
AOP 面向切面编程，把多个业务模块的公共行为封装成一个可重用的模块，通过动态代理的方式让业务模块具备这些功能；增加代码复用率，降低代码耦合度。

应用：
日志模块、事务管理、权限管理等

实现原理：
动态代理
动态代理的实现方法：
实现InvocationHandler接口，重写invoke方法；在Invoke方法里写增强处理的逻辑，这个公共代理类在运行的时候才能明确自己要代理的对象，同时可以实现该被代理类的方法的实现，然后在实现类方法的时候可以进行增强处理。

AOP实现的方式：
1.通过注解 @Aspect定义切面 @Before/@After/@Around(表达式) 在切入方法执行前后插入功能
2.通过Spring的API实现AOP
3.自定义类来实现AOP，不实现spring的自带的通知

发表于 2018-03-21 | 更新于 2018-03-28 | 分类于 面试

集合

Java中常用的数据结构

List: ArrayList、LinkedList
Set: HashSet
Map: HashMap、ConcurrentHashMap

ArrayList、LinkedList

ArrayList：基于动态数组实现 默认大小10 扩容1.5倍+1 线程不安全 CopyOnWriteArrayList或Vector
LinkedList：基于双向链表实现
区别：
1.随机访问元素时，ArrayList优于LinkedList；(因为LinkedList要移动指针)
2.新增或删除元素时，LinkedList优于ArrayList；(因为ArrayList要移动数据）
注意：Arraylist不能用正向for循环删除元素
